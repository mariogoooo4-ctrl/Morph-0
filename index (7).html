Instructions:

Using iOS Shortcuts:

1. Open the Shortcuts app on your iPhone.

2. Create a new Personal Automation (e.g., triggered daily at midnight).

3. Add an action: "Get Health Samples" (for Steps, Active Energy, Heart Rate).

4. Add an action: "Get Contents of URL".

5. Set the URL to your personal sync link above.

6. Set the method to POST.

7. Set the Request Body to JSON and add parameters like `steps`, `calories_burned`, `heart_rate` (using the Health Sample variables).

8. Disable "Ask Before Running".

Using Health Auto Export:

1. Download the Health Auto Export app from the App Store.

2. In the app, go to "Exports" and create a new custom export.

3. Configure the data you want to export (Steps, Active Energy, Heart Rate, etc.).

4. Set the export destination to "Webhook".

5. Paste your personal sync link into the Webhook URL field.

6. Configure the export frequency (e.g., daily).
